
HTML템플릿 제공 프로젝트

## Specs
+ [Svelte](https://svelte.dev/)
+ [SvelteKit](https://kit.svelte.dev/)

  
## Packages
+ svelteKit: code splitting등 고성능 웹 앱을 구축하기 위한 프레임워크이고, SSR을 지원합니다.
+ svelte: Svelte API를 사용하기 위한 핵심 패키지입니다.


## Build Setup
> 프로젝트 시작 전 패키지 설치
```
npm i
```
> 개발 서버 시작
```
npm run dev
```
> 빌드
```
npm run build
```
